## Emphasis

```
*Italic* or _Italic_
**Bold** or __Bold__
**_Bold and Italic_**
~~Strikethrough~~
```

*Italic* or _Italic_
**Bold** or __Bold__
**_Bold and Italic_**
~~Strikethrough~~

## Lists

### Unordered
```
- Item 1
- Item 2
  - Subitem 2.1
  - Subitem 2.2
```

- Item 1
- Item 2
  - Subitem 2.1
  - Subitem 2.2

### Ordered
```
1. First item
2. Second item
3. Third item
```

1. First item
2. Second item
3. Third item

## Code

Inline `code` has backticks around it.

    ```python
    # Code blocks use triple backticks
    def hello_world():
        print("Hello, World!")
    ```

```python
# Code blocks use triple backticks
def hello_world():
    print("Hello, World!")
```
